# React Weather Dashboard

React Weather Dashboard is a modern web application that allows users to search for any city and view real-time weather information.

The application fetches weather data from a public API and dynamically displays the results along with proper loading and error states.

🌐 **Live Website**
https://deepthip711.github.io/react-weather-dashboard

---

## Features

* Search weather by city name
* Real-time weather data using API
* Displays temperature and wind speed
* Dynamic loading state while fetching data
* Error handling for invalid city input
* Weather icons based on temperature
* Clean and responsive UI

---

## Technologies Used

* React
* JavaScript (ES6)
* HTML5
* CSS3
* Open-Meteo API

---

## How to Run the Project

Clone the repository

```bash
git clone https://github.com/deepthip711/react-weather-dashboard.git
```

Navigate to the project folder

```bash
cd react-weather-dashboard
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

Application runs at:

```text
http://localhost:3000
```
